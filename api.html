Gemini API Prompting

This short tutorial shows a minimal Python example that sends a text prompt to the Gemini (Google GenAI) API using the google-genai SDK. Each code line is explained to help students understand what is happening.

Full example code


# Install SDK
# pip install google-genai

from google import genai

# ✅ Initialize client with your API key
client = genai.Client(api_key="YOUR_GEMINI_API_KEY")

# Prepare prompt
model_name = "gemini-2.0-flash"
prompt_text = "Write a short note on RGUKT Basar."

# Generate response
response = client.models.generate_content(
    model=model_name,
    contents=prompt_text
)

# Print output
print(response.text)

Creates a client instance for communicating with the Gemini service. 9–10: 
# Prepare a prompt
model_name = "gemini-2.5-flash"
prompt_text = "Write a short note on RGUKT Basar."
Here we define two variables:
  • model_name — a string identifying which Gemini model to use. Different models have different capabilities and costs.
  • prompt_text — the natural-language instruction (the prompt) you want the model to complete. In this sample, it asks for a short note on RGUKT Basar.
These values can be adjusted depending on the task and model you choose.
13–17: 
# Call the API to generate content
response = client.models.generate_content(
    model=model_name,
    contents=prompt_text
)
This block sends the prompt to the Gemini model and stores the model's response in response.
  • client.models.generate_content is the SDK method that triggers content generation.
  • model= selects the model to use.
  • contents= supplies the prompt text. The SDK accepts other options too (like temperature, max output tokens, examples for few-shot prompting), but this minimal call focuses on required fields.
The method returns an object (often with attributes for the generated text, metadata, etc.). The exact structure may vary by SDK version — use print(dir(response)) if you want to inspect available fields.
20: 
# Print the result
print(response.text)
Prints the generated text to standard output. Many SDK response objects expose a .text or similar property containing the final generated string. If the SDK version you use returns a different field, inspect the response to find the appropriate attribute (e.g., response.output[0].content).

Note about authentication

The example uses genai.Client() without an explicit API key — this works only when credentials are available via environment variables or default application credentials. If you do not have credentials configured, create the client with an API key like this:

client = genai.Client(api_key="YOUR_GEMINI_API_KEY")

Replace "YOUR_GEMINI_API_KEY" with your actual key, or prefer using environment variables for better security.

Quick tips

  • Keep prompts clear and specific; the model follows the instruction you give it.
  • Use short test prompts while developing, then scale up once your code works.
  • Inspect the full response object during learning to see usage metadata (tokens used, latency, etc.).
  • Handle exceptions (network errors, rate limits) in production code.
